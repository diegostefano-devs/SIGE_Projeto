SIGE - Sistema Integrado de Gestão de Estoque

Sobre o projeto:
    Trabalho desenvolvido para o curso de Análise e Desenvolvimento de Sistemas - UNIFAN. Desenvolvido totalmente em Python, possui o objetivo de aplicar conceitos fundamentais da linguagem, como funções, listas, estruturas condicionais, laços de repetição, modularização e persistencia de dados. Concentrando o gerenciamento de clientes e produtos em um só ambiente.

Funcionalidades:

    Cadastro de Produtos
    Listagem de Produtos
    Exclusão de Produtos
    Cadastro de Clientes
    Listagem de Clientes
    Exclusão de Clientes
    Ponto de Venda
    Ponto de Reposição
    Relatório de Vendas por Cliente
    Relatório de Vendas Geral

Arquivos:
    menu.py: Responsável pela interface principal e navegação pelo sistema.
    funcoes.py: Contém as funções utilizadas na manipulação de dados
